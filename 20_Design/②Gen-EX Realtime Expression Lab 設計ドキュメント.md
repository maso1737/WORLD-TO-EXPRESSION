# **Gen-EX Realtime Expression Lab: 階層的知覚・表現教材 設計コンパニオン**

本書は、「触れば再現できる理解」を目的としたインタラクティブ教材シリーズ「Gen-EX Realtime Expression Lab」の基盤となる設計ドキュメントである。単なる知識の羅列やグラフィックデザインのテクニック解説ではなく、自然界の物理法則から人間の無意識の知覚、そして数理パターンがどのように視覚言語へ昇華され、最終的な表現（Expression）へと結実するかを7つの階層に体系化した。すべての要素は、Webアプリ（Three.js / GLSL）やRemotionといった数理・コードベースの教材において「操作可能なパラメータ」へ変換できるように定義されている。

## **1\. 全体アーキテクチャ・マップ（The Gen-EX Framework）**

物理世界から人間の表現に至る7つの階層構造、それぞれの核心となる問い、およびインタラクティブ教材における操作パラメータの対応関係を以下に定義する。

| 階層（LEVEL） | 核心的な問い | 主な学術的背景 | 教材化における「触れるパラメータ」の例   |
| :---- | :---- | :---- | :---- |
| **1\. PHYSICAL WORLD** | なぜ動く？ | ニュートン力学、運動学、流体力学 | 重力加速度(g)、摩擦係数(μ)、粘性、反発係数(e) |
| **2\. NATURAL PATTERNS** | なぜその形になる？ | 数理形態学、複雑系、自己組織化理論 | 分岐角度、反復回数（Fractal）、拡散率（反応拡散）、周波数・振幅（Noise） |
| **3\. PERCEPTION** | なぜ気づく？ | 知覚心理学、生態心理学、精神物理学 | コントラスト比、移動速度（Speed）、重心バランス、空間密度 |
| **4\. GESTALT** | なぜまとまって見える？ | ゲシュタルト心理学、認知科学 | 要素間距離（Proximity）、色・形状の合致度（Similarity）、同期ベクトル（Common Fate） |
| **5\. VISUAL LANGUAGE** | なぜ伝わる？ | デザイン理論、視覚伝達心理学 | 位置座標（Layout）、グリッドの分割数、リズム（周期性）、スケール比（Hierarchy） |
| **6\. ANIMATION 12 PRINCIPLES** | なぜ生きて見える？ | バイオロジカル・モーション、直感物理学 | 変形率（Squash & Stretch）、予備動作のフレーム数（Anticipation）、イージングカーブ |
| **7\. EXPRESSION** | どう表現する？ | メディアアート、インタラクティブ・デザイン | シェーダー（GLSL）、頂点変形、リアルタイムオーディオリアクティブ連動 |

## **2\. 各階層の深掘りとパラメータへの変換**

### **\[LEVEL 1\] PHYSICAL WORLD（なぜ動く？）**

生命やグラフィックが動き出す根源には、現実世界の物理法則が存在する。人間は進化の過程で「物理的に正しい挙動」を脳内にシミュレータとして持っており（直感物理学）、これの数値を意図的に狂わせることで、違和感や超現実的な感覚を生み出すことができる。  
**教材化のアプローチ:** 画面上のオブジェクトの質量、重力、摩擦をスライダーでリアルタイムに変更させ、「月面のような挙動」や「水中 のような挙動」を体感させる。

### **\[LEVEL 2\] NATURAL PATTERNS（なぜその形になる？）**

自然界が自律的に形成する数理的な規則性。美しさの根底にある幾何学的・動的なアルゴリズムを整理する。

* **フィボナッチ / 黄金比 (φ):** 植物の花弁の配列や螺旋構造に見られる、最も効率的な空間充填の数理。教材では「成長角度」をスライダーで操作し、137.5度（黄金角）になった瞬間に美しいグリッドが立ち上がる体験を作る。  
* **フラクタル（自己相似性）:** 全体と部分が同じ構造を持つ幾何学（例：マンデルブロ集合、コッホ曲線）。「再帰深度（Recursion Depth）」を変化させることで、単純な線が無限の複雑性へと変貌する様を見せる。  
* **ボロノイ（Voronoi Diagram）:** 複数の母点から最も近い空間を分割してできる細胞状のパターン（例：キリンの模様、乾燥した地面）。母点の「位置」や「数」を動かすことで、有機的な領地分割がインタラクティブに変化する。  
* **L-system（Lindenmayer system）:** 植物の成長プロセスを模した文法規則。「書き換えルール」と「分岐確率」をリアルタイムに変更し、線画がリアルな樹木へと進化する様子を可視化する。  
* **反応拡散（Reaction-Diffusion）:** 2つの化学物質が拡散し、反応することで生じる模様（例：シマウマのストライプ、熱帯魚の柄）。チューリング・パターンとも呼ばれる。拡散係数 *F*, *k* を微調整することで、ドット模様から波線、あるいは混沌状態へと相転移する境界をスライダーで探らせる。  
* **ノイズ（Perlin / Simplex Noise）:** 自然な揺らぎやテクスチャを表現するための連続的なランダム。純粋なホワイトノイズ（完全な無秩序）とは異なり、周波数（Frequency）と振幅（Amplitude）を操作することで、風の揺らぎや地形の起伏のような「有機的な動態」を生成する。

### **\[LEVEL 3\] PERCEPTION（なぜ気づく？）**

脳が感覚器官からの情報を受け取る、無意識かつ根源的な知覚プロセス。主に知覚心理学や生態心理学（ギブソンのアフォーダンス理論）の領域で扱われる。  
これらは、Web上の単一オブジェクトのプロパティを操作するだけで直感的に体感できる「感覚の最小単位」である。

* **安定 / 不安定:** 画面内の重心位置、底辺の広さによる心理的プレッシャー。  
* **重い / 軽い:** 加速度、バウンドの減衰率、あるいは色（明度）による重量感の錯覚。  
* **速い / 遅い:** 単位時間あたりの移動ピクセル数。フレームレートと時間知覚の相関。  
* **密集 / 疎:** 空間内の要素数とそれらが占める面積の比率（プレッシャーの強弱）。  
* **静的 / 動的:** 静止状態とベクトルの有無。ベクトルの向きによる視線誘導。  
* **緊張 / 解放:** 要素が画面の端（フレーム）に衝突しそうな限界状態（緊張）と、中央への収束（解放）。  
* **秩序 / 混沌:** 規則的な配列（予測可能）から、ランダム（予測不可能）へのグラデーション。  
* **大きい / 小さい:** 画面占有率、および周囲の要素との対比による相対的なスケール感。

### **\[LEVEL 4\] GESTALT（なぜまとまって見える？）**

人間の脳が、バラバラの視覚要素を1つの「意味のあるグループ」として統合する認識の法則。UI/UXデザインやモーショングラフィックスにおいて、情報の優先度を制御するコアロジックとなる。

| ゲシュタルト則 | 視覚的定義 | 現代UI/モーショングラフィックスへの応用   |
| :---- | :---- | :---- |
| **Proximity（近接）** | 距離が近いもの同士がグループに見える。 | UIのカードレイアウト、リストにおける項目とラベルの間白（Margin）の制御。 |
| **Similarity（類似）** | 色、形、大きさが似ているものがグループに見える。 | エラーボタンの一斉赤色化、特定カテゴリタグのカラーコーディング。 |
| **Continuity（連続）** | 直線や滑らかな曲線に沿った要素が繋がって見える。 | タイムライン UI の縦線、スクロールを誘導するプログレスバーや視線誘導線。 |
| **Closure（閉合）** | 閉じられていない不完全な形を、脳が補完して閉じた形とみなす。 | ミニマルなアイコンデザイン、ローディングスピナー（一部が欠けた円）。 |
| **Common Fate（共通運命）** | 同じ方向、同じ速度で動く要素が1つの群れに見える。 | メニュー展開時の子要素の一斉フェードイン、粒子（パーティクル）が文字に収束する演出。 |
| **Connectedness（連結性）** | 物理的な線や面で結ばれている要素が、最も強くグループと認識される。 | ノードベースUI（BlenderやComfyUI）、フローチャート、タブとコンテンツエリアの結合体。 |
| **Figure Ground（図と地）** | 視野が「主役（図）」と「背景（地）」に分離して認識される。 | モーダルウィンドウ表示時の背景のブラー（ぼかし）やダークアウト、ドロップシャドウによる浮遊感。 |

### **\[LEVEL 5\] VISUAL LANGUAGE（なぜ伝わる？）**

グラフィックデザインの構成要素。これらは「アートのセンス」ではなく、前述の「PERCEPTION」や「GESTALT」を駆動させるための**明示的な制御パラメータ**として再定義される。

* **Shape（形状）:** 境界線の数学的定義。円（安心・調和）、正方形（安定・信頼）、三角形（緊張・指向性）。角の丸み（Border-radius）は感情を操作する変数である。  
* **Line（線）:** 点の軌跡であり、ベクトルの指向性。太さ（Weight）は声の大きさに、実線/破線（Style）は確実性の度合いに対応する。  
* **Space（余白/マージン）:** 知覚の排他空間。「地」の面積をコントロールすることで、情報の重要度やエレガンスを操作する。  
* **Layout / Composition（配置・構図）:** 画面内の力の均衡。対称性（Static / 厳格）か非対称性（Dynamic / 解放）かを制御するアライメント変数。  
* **Hierarchy（階層性）:** 視線が動く順番の制御。文字サイズ比（Em/Rem）やコントラストを用いて、情報のスキャン速度をハックする。  
* **Rhythm（周期性・拍子）:** 要素の等間隔・不等間隔な反復。ユーザーのスクロール体験や時間軸上の心地よさをデザインする。  
* **Contrast（対比）:** 差分（Δ）。明度差、形状差、静動差。コントラストがゼロの時、情報は消失する。  
* **Color（色彩）:** HSL（色相・彩度・輝度）のパラメータ空間。感情を励起させる、または情報の分類コードとして機能する。

### **\[LEVEL 6\] ANIMATION 12 PRINCIPLES（なぜ生きて見える？）**

ディズニーが培った12の原則は、数理的には「物理法則の誇張（物理エンジンのハック）」と「脳の予測シミュレーションの充足」である。  
**パラメータ化の例:**  
・*Squash and Stretch（押し潰しと引き伸ばし）:* 速度ベクトル *v* に応じて、体積を一定（*x × y × z \= k*）に保ったままアスペクト比を変形させる関数。  
・*Anticipation（予備動作）:* 主運動の前に、逆方向のベクトルを一定時間与えるディレイ制御。  
・*Timing and Spacing（タイミングと配置）:* イージング関数のベジェ曲線（Bezier Curve）の制御点（*cp1, cp2*）の操作。

### **\[LEVEL 7\] EXPRESSION（どう表現する？）**

最下層から積み上げてきたすべての階層（物理・数理・知覚・認識・デザイン・アニメーション）を統合し、リアルタイムコード、音楽、映像、3D空間へと昇華した、ユーザーが最終的に目にする「体験そのもの」。

| **9\. ROTATE** | PHYSICAL WORLD / VISUAL LANGUAGE | 回転、角度、指向性の変化。動的均衡の破壊。 | 回転角（θ）、または「角速度（ω）」。静的グリッドを5度傾けただけで生まれる動的な緊張感。 |
| **10\. EMERGENCE** | NATURAL PATTERNS / GESTALT | 創発。個の単純ルールから全体の意味が立ち上がる現象。 | Boidsアルゴリズム（群れのシミュレーション）における「分離・整列・結合」の3パラメータ調整。 |

## **5\. Webアプリ・Remotion化しやすい教材テーマ候補（実装アイデア）**

コードベース（Three.js, GLSL, GSAP, Remotion）で即座にプロトタイプ化でき、「触れば1秒でわかる」インタラクティブ教材の具体的な企画案。

### **① 教材テーマ名: 「反応拡散（Turing Pattern）の相転移シミュレータ」**

* **ターゲット階層:** NATURAL PATTERNS → PERCEPTION（秩序と混沌）  
* **実装仕様（Web / GLSL）:** フラグメントシェーダー（GLSL）を用いて、GPU上で毎フレーム反応拡散方程式を計算。画面横に「拡散率U」「消滅率V」のスライダーを配置。  
* **体感できること:** スライダーを数ピクセル動かしただけで、画面全体が「均一なグレー（無）」から「シマウマ模様（秩序）」、そして「スポットパターン（細胞）」へと劇的に相転移する境界を自らの手で探り当てることで、自然界が形を作るメカニズムを体感する。

### **② 教材テーマ名: 「ゲシュタルト則をハックする同期パーティクル」**

* **ターゲット階層:** PERCEPTION → GESTALT（Common Fate / Proximity）  
* **実装仕様（Three.js / GSAP）:** 画面にランダムに浮遊する300個のドットを配置。  
  * スライダーA（Proximity）: ドット同士の距離を縮める。ある閾値で「星座」のように塊として認識される。  
  * スライダーB（Common Fate）: 別々の位置にあるドット群に、全く同じ「サイン波の上下ベクトル」を与える。動いた瞬間に、離れていても「同じグループ」に見える。  
* **体感できること:** 静止画（グラフィック）ではバラバラに見えるものが、「動きの同期」という時間軸パラメータが加わった瞬間に、脳内で強烈なグループ化を起こす「モーショングラフィックスの本質」を体感する。

### **③ 教材テーマ名: 「ディズニー12原則の数理：ベジェ曲線と生命感」**

* **ターゲット階層:** VISUAL LANGUAGE → ANIMATION 12 PRINCIPLES  
* **実装仕様（Remotion or Webコード）:** 単純な四角形が左から右へ移動するアニメーション。  
  * スライダー1（Squash & Stretch）: 移動速度に応じて四角形がどれだけ引き伸びるかの変形率。  
  * スライダー2（Anticipation）: スタート前に逆方向に沈み込むフレーム数。  
  * CSSの transition-timing-function（Cubic-Bezier）の制御点を画面上のグラフで直接いじる。  
* **体感できること:** 等速（Linear）では「ただのバグった四角形」に見えるコードが、数理的な「タメ・ツメ」と「変形」を加えた瞬間に、生き物としての「感情や質量」を帯びる変化の瞬間を目の当たりにする。

## **6\. 学術的裏付けのある参考分野・キーワード**

教材の解説テキストや、今後のアルゴリズム実装のベースとなる学術的リファレンス群。

* **数理生物学・形態形成論:** アラン・チューリングの論文「The Chemical Basis of Morphogenesis（形態形成の化学的基礎）」。反応拡散系のルーツ。  
* **複雑系・自己組織化（Self-Organization）:** 局所的な単純ルール（例：セル・オートマトン、ライフゲーム）が、全体として予測不可能な高次パターン（カオスからの秩序）を生み出す数理。  
* **生態心理学（Ecological Psychology）:** ジェームズ・J・ギブソンの「アフォーダンス理論」。環境が生物に提供する知覚的価値。人間はグラフィックを見るのではなく、グラフィックが持つ「物理的気配」を探索しているという視点。  
* **精神物理学（Psychophysics）:** ウェーバー・フェヒナーの法則（刺激の強さと人間の感覚の大きさは対数関係にある）。スライダーの数値をリニアではなく対数（Logarithmic Scale）で変化させるべきだという実装上の裏付け。  
* **視覚神経科学:** 脳の一次視覚野（V1野）におけるエッジ検出器（ガボール・フィルタ）。人間が「線（Line）」や「コントラスト（Contrast）」に無意識に反応してしまう神経細胞レベルの仕組み。
